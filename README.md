# 3SIG Platform

> **Traffic × Product × AI** | News Intelligence Pipeline

## Overview

3SIG는 뉴스 데이터를 수집, 분석하여 트래픽(블로그/소셜), 지식(온톨로지), 제품 아이디어로 변환하는 인텔리전스 플랫폼입니다.

## Pipeline

```
Raw Data (HN, Reddit, X) -> Analysis -> Traffic / Knowledge / Product
```

## Tech Stack

- Next.js 14, Tailwind CSS
- Supabase (PostgreSQL)
- Vercel
- Claude + MCP

## Quick Start

```bash
npm install
cp .env.example .env.local
npm run dev
```
